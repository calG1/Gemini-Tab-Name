# Gemini Tab Renamer

This Userscript automatically updates your browser tab title to match the current chat title in Google Gemini.

## Installation

1.  **Install Tampermonkey**:
    *   [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
    *   [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
    *   [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
    *   [Safari](https://apps.apple.com/us/app/tampermonkey/id1482490089)

2.  **Add the Script**:
    *   Click on the Tampermonkey icon in your browser toolbar.
    *   Select **"Create a new script..."**.
    *   Delete any default code in the editor.
    *   Copy the entire contents of `gemini_tab_renamer.js` from this repository.
    *   Paste the code into the Tampermonkey editor.

3.  **Save**:
    *   Press `Ctrl+S` (or `Cmd+S` on Mac) or go to **File > Save**.

4.  **Usage**:
    *   Open [Google Gemini](https://gemini.google.com/).
    *   The tab title should now reflect the active chat's name.
